Naming Rules Cheat Sheet

Common Cations

Fixed-charge metals never need a Roman numeral. Variable-charge (transition) metals always do.

Common Anions

Monatomic anions end in -ide.

Common Polyatomic Ions

-ate has one more oxygen than -ite. Memorize NO₃⁻ and SO₄²⁻ as anchors.

Covalent Prefixes

Used for nonmetal + nonmetal compounds. Drop the final "a"/"o" before another vowel (e.g. monoxide). Never use "mono-" on the first element.
